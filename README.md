# 天津大学本科生毕业论文模板

## 注意
* 首先声明，本人并未承诺该模板是完美无缺的，如果有任何由于使用本模板导致的毕业问题，请自行承担后果，切记，工具的目的是为了减少工作量，而非创造工作量。

## 使用方法
* 如果你拥有overleaf升级计划，那你完全可以在overleaf上使用本模板进行论文撰写，但是如果你不是的话，可能会因为参考文献问题导致编译时间超时，所以建议你可以使用本地的latex环境进行编译使用。
* 本人在模板中已经撰写了部分使用的例程，如果你拥有latex的使用经验，上手时间绝不会超过半小时；即使你没有，AI时代下心流状态也会让你在1小时左右完成基础的教程学习，相信我，内容远胜过形式！
* 补充说明：封面部分本模板并未进行设计(过于复杂且出错概率太高)，采用word撰写完毕，自行补充pdf到cover.pdf中，剩余部分均可在latex中自行解决。

## 适用人群
* 如果你是极度厌烦复杂的格式要求的人，渴望能专注于写作本身，且从未接触过latex，那么尝试利用这个机会了解这个极其好用的工具将会为你的科研或者自我创作起到正向的作用。
* 如果你是资深的latex专家，毫无疑问，你将省去撰写模板的时间。
